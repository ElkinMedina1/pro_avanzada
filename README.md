# Gestión de Estudiantes

## Descripción
Es un proyecto en Python que maneja una lista de estudiantes. Cada estudiante tiene nombre, edad, estado, materias y notas.

Primero, el programa revisa que el estudiante tenga al menos 3 materias y que las notas estén entre 0.0 y 5.0.  
Si todo está bien, calcula el promedio, identifica la materia con la mejor nota, la peor nota y determina si el estudiante aprueba.

Un estudiante aprueba si su promedio es 3.0 o más.

## Cómo correrlo
Abre la terminal, ve a la carpeta donde está el archivo y escribe:

```bash
python ejercicio_1.py
